DevOps CI/CD Core

Automated Static Portfolio CI/CD Pipeline

A production-style CI/CD pipeline built from scratch to automate the packaging and deployment of this responsive, interactive portfolio website.

Every time a commit is pushed to the GitHub repository, a GitHub Webhook securely signals the Jenkins server hosted on AWS EC2. Jenkins immediately triggers a multi-stage Declarative Pipeline that lints assets, builds the static bundle using Vite, packs a secure Alpine-Nginx Docker image, pushes it to DockerHub, cleans the old running container on EC2, and runs the new deployment with zero-downtime scripting.
